## 严重程度定义

| 严重程度 | 描述 |
|----------|------|
| 阻塞 | 导致系统无法正常运行，核心功能失效，必须立即修复 |
| 严重 | 影响系统主要功能，导致部分功能无法正常使用 |
| 一般 | 影响系统次要功能，或功能可用但体验不佳 |
| 轻微 | 不影响系统功能，仅为代码质量或文档问题 |

## 解决状态定义

| 状态 | 描述 |
|------|------|
| 待解决 | 问题已发现，尚未开始修复 |
| 解决中 | 问题正在修复过程中 |
| 已解决 | 问题已修复，等待验证 |
| 已验证 | 问题已修复并通过验证 |
| 已关闭 | 问题已解决并关闭 |
| 延后 | 问题暂时不修复，延后处理 |
| 拒绝 | 问题不成立或无需修复 |
